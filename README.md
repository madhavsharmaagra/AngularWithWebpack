# AngularWithWebpack
# Project Title
Demo Project - Angular 4 application with Webpack. 

This repo is completely for Learning purpose. 

## Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Installing

```
npm install
```
### To see some fun, run
```
npm start
```
